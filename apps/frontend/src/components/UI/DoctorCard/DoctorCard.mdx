# DoctorCard

The DoctorCard component provides primitive building blocks for creating various doctor cards. It is created using
`Compound component` pattern to make the component composable and allow the placement of other components inside it.

<Canvas of={DoctorCardStories.Default} />

The code:

```ts
<DoctorCard>
        <DoctorCard.Image url='https://thispersondoesnotexist.com/' />
        <DoctorCard.Name>John Doe</DoctorCard.Name>
        <DoctorCard.Specializations specializations={[{ id: '1', name: 'Therapist' }]} />
        <DoctorCard.PayrateLabel payrate={75} />
        <DoctorCard.Tags tags={['Top doctor']} />
        <DoctorCard.Rating rating={4.5} reviewsCount={10} />
</DoctorCard>
```
